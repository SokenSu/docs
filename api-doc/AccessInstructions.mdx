- **Access overview：**

This reference provides a comprehensive overview of the OpenAI API features. The API follows  RESTful principles, and uses OAuth 2.0.
1. You need to request appId and appSecret and signSecret by email.
2. The openAI emails you appId and appSecret and signSecret.
3. You authenticate with appId and appSecret and signSecret.

- **OpenAPI Access diagram：**
1. You can get customer user info and station info and device info, and control the device.

Device control

1. Use appId and appSecret to call Get token for authentication
2. You can execute the Device execute command to send the control device command, and the status of the control device command is executing
3. Polling Get Device command event Obtains the result of the control device command until the status of the control device command is success or fail
4. If you configure a webhook, the webhook notifies you when the control device command is success
